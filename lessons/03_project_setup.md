# 第3课：项目结构、运行与提交实践
目标：理解当前结构，能运行游戏，掌握工作区/暂存区/提交。

1. 项目结构说明
   - 俄罗斯方块/：index.html、style.css、src/main.js、src/board.js、src/tetromino.js
   - lessons/：课程讲义
   - cheatsheets/：速查表
2. 运行游戏
   - 双击 俄罗斯方块/index.html 即可本地运行
3. 工作区暂存区提交
   - 修改 俄罗斯方块/style.css（例如按钮颜色）
   - 查看状态：git status
   - 暂存并提交：
     - git add 俄罗斯方块/style.css
     - git commit -m "style: 调整按钮颜色"
4. 使用 GitLens
   - 在编辑器中打开被改文件，查看行级信息和文件历史
   - 比较当前提交与上一次提交的差异
